## 題庫 JSON 格式說明

題目檔 (`*-quiz.json`) 結構：
```json
{
  "id": 1,
  "question": "題目內容",
  "options": { "A": "選項A", "B": "選項B", "C": "選項C", "D": "選項D" }
}
```

答案檔 (`*-answers.json`) 結構：
```json
{
  "id": 1,
  "answer": "C",
  "explanation": "解析說明"
}
```
