# 元素週期性質趨勢 - 規則要領與解題技巧

## 一、週期表基本方向

```
        ← 週期表向左

   ↑    Li Be B  C  N  O  F  Ne
   向    Na Mg Al Si P  S  Cl Ar
   上    K  Ca ...
```

記住兩個主要方向：
- **同一週期（橫向）**：由左至右
- **同一族（縱向）**：由上至下

| 性質 | 同週期（→右） | 同族（↓下） | 記憶口訣 |

| **原子半徑** | 減小 ↓ | 增大 ↑ | 「右小下大」 |

| **非金屬性** | 增大 ↑ | 減小 ↓ | 「右上最非」 |

## 三、各性質詳解與原理

### 1. 原子半徑

**趨勢**：右小、下大

**原理**：
- 同週期向右：質子數增加 → 有效核電荷增加 → 拉緊電子 → 半徑變小
- 同族向下：電子層數增加 → 半徑變大

**解題技巧**：
```
比較 Na, Mg, K 的原子半徑大小？

步驟：
1. 找位置：Na(第3週期1A), Mg(第3週期2A), K(第4週期1A)
2. 同週期：Na > Mg（向右變小）
3. 同族：K > Na（向下變大）
4. 答案：K > Na > Mg
```

### 2. 離子化能（游離能）

**定義**：移除一個電子所需的最小能量

**趨勢**：右高、下低（惰性氣體最高）

**特殊例外**：
- **Be > B**：B 的 2p¹ 較易移除
- **N > O**：N 的 2p³ 半滿穩定
- **Mg > Al**：Al 的 3p¹ 較易移除

**解題技巧**：
```
第一離子化能：N > O > C

原因：N 的 2p³ 為半滿組態，特別穩定
```

### 3. 電子親和力

**定義**：原子獲得一個電子釋放的能量

**趨勢**：鹵素（F, Cl, Br, I）最大，Cl > F

**為何 Cl > F？**
- F 原子太小，電子間排斥力大

### 4. 電負度

**趨勢**：F 最大（4.0），Cs 最小（0.7）

**常用電負度**：
```
F(4.0) > O(3.5) > Cl(3.0) ≈ N(3.0) > Br(2.8) > C(2.5) > H(2.1)
```

## 四、離子半徑比較技巧

### 規則整理：

| 比較情況 | 規則 | 範例 |
|----------|------|------|
| 同元素陽離子 | 電荷越大，半徑越小 | Fe²⁺ > Fe³⁺ |
| 同元素陰離子 | 電荷越大，半徑越大 | O⁻ < O²⁻ |
| 等電子離子 | 質子越多，半徑越小 | O²⁻ > F⁻ > Na⁺ > Mg²⁺ > Al³⁺ |
| 陽離子 vs 原子 | 陽離子 < 原子 | Na⁺ < Na |
| 陰離子 vs 原子 | 陰離子 > 原子 | Cl⁻ > Cl |

### 等電子系列（10個電子）解題：
```
O²⁻, F⁻, Ne, Na⁺, Mg²⁺, Al³⁺

質子數：8, 9, 10, 11, 12, 13
半徑：  大 ─────────────→ 小

答案：O²⁻ > F⁻ > Ne > Na⁺ > Mg²⁺ > Al³⁺
```

## 五、快速解題口訣

### 萬用口訣：「氟右上，銫左下」

```
              非金屬性最強
                   ↗
    金          F
    屬   ↖
    性      Cs
    最
    強
```

### 性質分類記憶：

**「右上派」**（向右上角增大）：
- 離子化能、電子親和力、電負度、非金屬性

**「左下派」**（向左下角增大）：
- 原子半徑、金屬性

## 六、常見題型與解法

### 題型一：排列大小

```
題目：將 S, Cl, Ar, K 按原子半徑排列

解法：
1. 位置：都在第3週期附近
2. S, Cl, Ar 同週期 → Ar < Cl < S（向右變小，但Ar例外要小心）
3. K 在第4週期 → K 最大
4. 答案：K > S > Cl > Ar
```

### 題型二：判斷金屬/非金屬性

```
題目：比較 Na, Mg, Al 的金屬性

解法：
1. 同週期向右，金屬性減弱
2. 答案：Na > Mg > Al
3. 驗證：與水反應活性 Na > Mg > Al ✓
```

### 題型三：預測化合物性質

```
題目：NaCl 與 MgO 哪個熔點較高？

解法：
1. 離子化合物熔點 ∝ 電荷乘積 / 離子距離
2. MgO：(+2)(-2) = 4，離子小
3. NaCl：(+1)(-1) = 1，離子大
4. 答案：MgO 熔點較高
```

## 七、例外情況整理

| 例外 | 正常預測 | 實際情況 | 原因 |
|------|----------|----------|------|
| 電子親和力 Cl > F | F > Cl | Cl > F | F 太小，排斥力大 |
| 離子化能 O > N | O > N | N > O | N 的 p³ 半滿穩定 |
| 離子化能 B > Be | B > Be | Be > B | Be 的 s² 全滿穩定 |
| 原子半徑 Ga ≈ Al | Ga > Al | Ga ≈ Al | d 軌域收縮效應 |
